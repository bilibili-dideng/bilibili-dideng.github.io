# 隐私政策

← 返回索引

**生效日期：2025年4月5日**  
**适用范围：CA Library IDE 桌面开发环境**

## 1. 我们收集哪些信息

CA Library IDE 是一款本地运行的桌面开发工具，**不收集、不上传、不共享任何用户数据**。  
我们尊重您的隐私，所有操作均在本地完成。

| 权限/功能    | 用途说明                        | 是否联网 | 可关闭性      |
|----------|-----------------------------|------|-----------|
| 读写本地文件系统 | 打开、保存 `.json` 拓展包文件，读取配置与日志 | 否    | 用户可自主选择路径 |

> ⚠️ **重点声明**：  
> - **我们不会收集任何个人身份信息（PII）**，包括但不限于姓名、邮箱、设备标识、IP 地址等。  
> - **无广告 SDK、无统计分析组件、无遥测（Telemetry）服务**。  
> - 所有项目文件、配置、日志均**仅保存于您的本地设备**，您完全掌控数据。

## 2. 数据存储与共享

- **本地优先**：所有拓展包、配置文件、日志均存储在您指定的本地目录中。
- **零上传**：本软件不会将任何数据上传至服务器（因为我也没有服务器）。
- **零共享**：不会与任何第三方（包括 CA Reforged 官方）共享您的数据。
- **彻底删除**：卸载软件并删除项目文件夹后，所有数据即被清除。

## 3. 与 CA Reforged 的关系

- CA Library IDE 为 **CA Reforged**（命令助手 Reforged 版）生态的第三方开发工具。
- 本软件由 **地灯dideng** 独立开发，非南鸢晨星或 ProjectXero 官方出品。
- 我们遵循与 CA Reforged 一致的隐私理念：**最小权限、本地优先、透明可信**。

## 4. 未成年人

未满 18 周岁用户请在监护人指导下使用本软件。本软件不涉及用户注册或数据收集，但建议监护人了解其用途。

## 5. 政策更新

若未来功能调整导致本政策需变更，我们将在以下位置提前公示：
- GitHub 仓库公告（https://github.com/bilibili-dideng/）
- 软件启动时的更新日志

继续使用即视为您接受更新后的政策。
