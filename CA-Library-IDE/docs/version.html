版本更新记录

更新重要度版本排序如下

  1. 更新重要度 < 190 : 修订号+1
  2. 更新重要度 < 320 : 次版本号+1
  3. 更新重要度 >= 320 : 主版本号+1

更新重要度计算：((((加入功能数 * 2) + (功能子项数 * 0.9)) + (移除功能数 * 1.5) + 优化功能数 * 1.2) * 1.1)² （保留2位小数）
