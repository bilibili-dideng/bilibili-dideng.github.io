核心功能

📝

行号与当前行高亮

清晰显示每一行代码，当前行自动高亮，便于定位与调试。

🎨

主题系统

支持通过拓展包加载自定义主题，适配不同视觉偏好。

⚡

CA 关键字快速插入

一键插入常用 CA 命令与语法结构，减少手动输入错误。

⇥

智能自动缩进

根据代码结构自动缩进，保持代码整洁规范。

📊

结构化视图

左侧显示 JSON/脚本结构树，快速跳转到目标节点。

📁

文件树管理

多文件项目支持，可视化文件目录，方便管理拓展包资源。

🔧

JSON 格式化与校验

自动格式化 JSON，语法错误实时提示，确保数据正确。

🧩

拓展包扩展功能

通过拓展包可增加新主题、新关键字模板，实现无限扩展。

关于 CA Library IDE

本项目专为 CA 拓展包 开发者设计，旨在解决原生编辑器功能不足的问题。

我们支持完整的 CA 语法体系，并通过插件化架构，允许社区开发 主题包 与 关键字模板包，持续扩展 IDE 能力。

目前项目处于内测阶段，暂不开放源码，但我们欢迎开发者申请试用并提供反馈。
